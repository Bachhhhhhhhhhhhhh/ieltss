# IELTS Mastery

Bilingual IELTS preparation platform (EN/VI) — mock tests, smart practice, lookup knowledge base, and progress tracking.

## Features

- 20 full IELTS mock tests (Listening, Reading, Writing, Speaking)
- Skill practice with vocabulary flip cards and grammar drills
- Lookup & Knowledge Base with smart search, voice input, study deck, and vocab quiz
- Progress dashboard with analytics and achievements
- Bilingual UI (English / Vietnamese)

## Quick start

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start dev server |
| `npm run build` | Production build |
| `npm run lint` | Run ESLint |
| `npm run preview` | Preview production build |

## Stack

- React 19 + TypeScript + Vite
- Tailwind CSS + Framer Motion
- Recharts
